# OntologyOps Examples

This directory contains runnable examples demonstrating OntologyOps capabilities.

## Quick Start

```bash
# From project root
pip install -e .
python examples/quickstart.py
```

## Examples

- **quickstart.py** - 5-minute demo of all features
- **version_control_demo.py** - Semantic version control
- **testing_demo.py** - Testing framework
- **deployment_demo.py** - Deployment workflow
- **monitoring_demo.py** - Monitoring integration

## Sample Ontologies

The `ontologies/` directory contains sample OWL files for experimentation.
