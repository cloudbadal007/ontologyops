# Installation

## pip

```bash
pip install ontologyops
```

## From Source

```bash
git clone https://github.com/cloudbadal007/ontologyops.git
cd ontologyops
pip install -e .
```

## Dependencies

- Python 3.9+
- rdflib >= 7.0
- networkx >= 3.0
- requests >= 2.28
- pyyaml >= 6.0

Optional for monitoring:
- prometheus-client >= 0.16
